⚙️ Cómo conectar tus datos
1
Crea un Google Sheet con las columnas: Date · Country · Inn Code · WMSR · Count Checkin · Msg Response · No Response · Total Time · Welcome Sent
2
Publica el Sheet: Archivo → Compartir → Publicar en la web → CSV. Copia la URL que termina en /pub?gid=0&single=true&output=csv
3
Pega la URL abajo y haz clic en "Cargar datos". El dashboard validará las fechas antes de mostrar los KPIs.
